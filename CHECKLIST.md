# Checklist de livraison client — AbiWeb

À cocher avant d'envoyer le lien de prévisualisation, puis avant la mise en ligne finale.

## Contenu
- [ ] Tous les `[PLACEHOLDERS]` de `site-content.js` sont remplacés
- [ ] Logo récupéré et intégré (ou nom stylisé si pas de logo)
- [ ] Photos reçues et intégrées (galerie, hero, etc.)
- [ ] Couleur primaire (`couleurPrimaire`) ajustée à l'identité du client
- [ ] Coordonnées (adresse, téléphone, email, horaires) vérifiées avec le client

## SEO
- [ ] `seo.titre`, `seo.description`, `seo.motsCles` renseignés
- [ ] `<title>` et `<meta name="description">` de base dans le `<head>` cohérents avec `seo`
- [ ] Compte Google Search Console créé pour le domaine du client
- [ ] `seo.googleSearchConsole` renseigné avec le code de vérification

## Analytics & RGPD
- [ ] Propriété GA4 créée pour le client (ou décision explicite de ne pas en mettre)
- [ ] `seo.googleAnalyticsId` renseigné (laisser vide si pas de tracking)
- [ ] Bandeau cookies testé (Accepter / Refuser) → vérifier dans l'onglet Réseau que GA ne se charge qu'après "Accepter"
- [ ] `mentionsLegales` renseigné (éditeur, adresse, hébergeur, SIRET si applicable)

## Technique
- [ ] Domaine connecté (ou sous-domaine gratuit confirmé avec le client)
- [ ] Site testé sur mobile (responsive)
- [ ] Tous les liens externes (réseaux sociaux, paiement, réservation) vérifiés
- [ ] Formulaire de contact testé (email bien reçu)

## Livraison
- [ ] Lien de prévisualisation envoyé au client
- [ ] Validation client reçue + solde encaissé
- [ ] Site mis en ligne
- [ ] Guide de mise à jour (modifier textes/photos) remis au client
- [ ] Rappel J+7 programmé (garantie retouches)
